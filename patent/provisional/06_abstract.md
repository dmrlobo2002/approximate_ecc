# ABSTRACT

A method and system for error correction encode a data block by applying a keyed, invertible permutation to bit indices — implemented as a multi-round Feistel network using a cryptographic hash function as the round function — to scatter errors uniformly across a two-dimensional (2D) grid. The grid comprises N rows and N columns, where N is approximately the square root of the block length L. Hash digests are computed over non-overlapping groups of rows and columns of the grid and stored as error-correction metadata alongside the data block, yielding overhead that scales as O(1/√L). During decoding, digests are recomputed over a received block arranged into the same grid, mismatched hash nodes are identified, and a constraint graph is constructed in which nodes correspond to hash computations and edges connect nodes with overlapping bit coverage. A greedy, constraint-guided search then iteratively selects mismatched nodes in order of ascending covered-bit count, enumerates bit-flip combinations within candidate indices — excluding bits pinned by already-matched neighboring nodes — and commits the flip combination that maximizes total matched nodes. This approach corrects random and burst errors with equivalent efficacy and achieves lower metadata overhead than algebraic codes such as BCH at equivalent correction capability, without requiring Galois field arithmetic.
