# ABSTRACT

A method and system for error correction encode a data block by applying a keyed invertible permutation — implemented as a multi-round Feistel network using a cryptographic hash function as a round function with cycle-walking for arbitrary domain sizes — to scatter burst and random bit-flip errors uniformly across a two-dimensional grid of N rows and N columns, where N is approximately the square root of the block length L. Hash digests computed over non-overlapping row groups and column groups of the grid are stored as error-correction metadata, yielding overhead that scales as O(1/√L). During decoding, digests are recomputed over the received block in the same grid, mismatched hash nodes are identified, and a constraint graph is constructed wherein nodes represent digest computations and edges connect nodes sharing bit coverage. A greedy, constraint-guided search iteratively selects mismatched nodes in order of ascending covered-bit count, enumerates bit-flip combinations within candidate indices while excluding bits pinned by matched neighboring nodes, and commits the combination that maximizes total matched nodes. The approach corrects random and burst errors with equivalent efficacy, achieves lower overhead than BCH codes at equivalent correction capability, and requires no Galois field arithmetic.
